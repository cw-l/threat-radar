## Demo

You can use this Thoughtworks [sample data](https://raw.githubusercontent.com/cw-l/threat-radar/refs/heads/master/data/sample.csv) now to see the Threat Actors Radar in action at [TAR](https://cw-l.github.io/threat-radar).

## How does this work?

Use TAR to track relevant Threat Actors (TAs). TAs are categorised by geography and industries. Each blip represents a TA with writeup on incidents attributed to or owned by TA.
